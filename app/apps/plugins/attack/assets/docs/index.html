Description
Plugin Attack keeps the system secure against potential computer attacks that exist in the network, with this plugin keep data safe system.

 
Configuration
After activating the plugin, select the link "Settings" to edit the characteristics of the plugin
Configure the GET and POST requests

  • GET max requests per time
  • GET time request [seconds]
  • POST max requests per time
  • POST time request [seconds]
  • Ban time [minutes]
  • Ban message

Then click on Submit to save the changes
Message comes out when some of the settings of the plugin is exceeded and the site is blocked by the time settings.
